怎么赚100万

发表于 2017-03-21 | | 阅读次数

这一篇文章，让自己认清楚现实，也为自己的将来谋一条出路，可能很长。

背景

地点：北京
年龄：22
职业：程序员
状态：离职在家（大四未毕业，准备毕业论文）

目标

挣100w，这里很简单，就是研究，在现状下，如何利用好自己的资源，挣到人生的100w。当然，这个要有一个尽量短的时间，

博主定下的目标是：2年内挣100w。

计划

好，目标有了，我们就是做出一个计划，然后实施就行了！

首先，做计划之前，具体化目标：

目标具体化

两年内挣100w，平摊就是1年50w，今年是2017年

粗略统计了一下，按照上述的计算结果，可以得知，
1.每天都能挣到钱，而且稳定的情况下，每天都必须达到1370元
2.如果你是打工，只在工作日能赚钱，那么每天都必须赚1916元
3.按月计算，则是每个月42000元

但是，博主要求的不是挣100w，而是结余100w(税后)。

那么假设博主每个月租房，吃饭，各种开销，按照6000元计算，帝都的房子一个月2500，，生活费2500，其他开销1000，不过分吧，
然后还要求是税后，就按照税后48000来计算

在网上找了个税计算器，计算得出
64000 税后 48330 (不包括五险一金)

初步结论:
1.每个月，基本要挣65000左右(稳定的情况)
2.不稳定情况下，峰值按2倍计算，需要达到13w/月

打好你手中的牌

一个没有背景，没有深厚的知识积累，在这种情况下，如何创造每个月65000或者峰值13w的价值。
那么，我们首先需要知道，我们可以利用的资源有哪些。博主这样的人，手上就是一副烂牌，如何能够打成一副好牌，创造价值呢。
下面看牌。

好了，分析完自己手里的资源。感觉真tm绝望。

首先。一条条来。
1.最好的选择是 原公司经理，和我很熟，他手头也有资源，
2.我最大的资本是我这个人本身，其他并没有什么太值得利用的。
3.爸妈的血汗钱，拿来创业，这是一个选择，但是你会成功吗。
4.好好学习，证明自己的价值？
5.有一个好的方案，2年时间实现，然后融资？

提出方案

提出假设和方案
1.找同学，家人出资伙同创业，2年100w不是梦，但是，我现在的知识储备能够创业成功吗？ 风险指数–90%
2.稳妥点，找家人要资金，开个店面，生意好能有2,30w的收益每年。3年挣够100w。风险指数–50%
3.买彩票。风险指数–99.99%
4.有一个想法，然后实现，融资创业 风险指数–80%
5.进入一个小公司，创业成功，分红数十万 –90%
…

还有很多这类方法。但是。真正能选择的不多。我需要的是一个可以长远发展的办法。
1.不能打工拿死工资。假设进入了bat，华为，10年后拿分红，年薪和奖金合计，100w一年，但是，10年青春，你愿意吗！！！—我不愿意。
2.打工非我所愿，无法实现财务自由，那么就只剩下创业！。

创业

创业环境。

  1. 技术创业:
    1. vr突破
    2. 机器学习普及
    3. 电池技术突破
    4. 能源行业
    5. 金融行业
  2. 做生意(不感兴趣，还是希望做计算机专业)
    1. 服装生意
    2. 电脑周边
  3. 资金来源
    1. 家人
    2. 朋友合伙
    3. 融资（需技术创业）

在上述选择中。我选择技术创业。
3个问题。

  1. 技术从何来
    1. 努力学习 80%无法突破
    2. 没了。从天上来。99%不可能
  2. 好的想法从何而来
    1. 纵观国内外差异，参考国外成熟的技术，在国内研究突破，创业成功。 50%
    2. 纵观科技史,社会学，发现新时代的需求。 20%
    3. 其他人提供，合伙创业 10%
      3.假设技术和想法都有了，你如何创业
    4. 融资。
    5. 创业起步资金从哪来，(家人和同学)
    6. 降低创业起步资金的要求。自己给自己打工，加上朋友

最后一个你觉得比较可行的方案：
1.看国内外差异，寻找突破点。
2.针对性的学习技术，为将来创业做准备
3.拿出具体可行的创业方案，不是空想。

问题:
创业你凭什么拉倒融资
你怎么发现国内外差异点和突破点
你怎么知道这个想法可行
你怎么知道这个想法没有人做
你怎么确保成功。
你怎么控制风险和成本
失败了如何
成功了如何
你知道融资的过程吗
你知道创建公司的过程吗
你的同学家人会给你钱吗
你打算用几个月来做这个事情
你熟悉公司的业务吗
你熟悉公司的流程吗
你知道你需要什么技术，欠缺什么人，该如何去做吗

具体的创业方案呢
用什么想法，几个月达到什么目标，最后是什么前景，技术如何变现，如何找到市场，如何获取用户，怎么控制风险。怎么解决困难，怎么按照预期完成计划。
