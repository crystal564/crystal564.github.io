在centOS 7 安装hadoop单机环境教程

首先，在虚拟机VMware上安装cent os 7，centos7和之前的版本安装步骤相同，命令不同，因为centos7的一些命令不同了。

安装虚拟机的过程不多说，博主的环境是centos7最小化安装，进去之后就是命令行，因为之后打算用secureCRT做链接，所以没有安装图形化界面。

首先，我们需要在root用户下安装hadoop，最好在root用户下安装，博主曾经尝试过普通用户安装，结果用户权限搞得乱七八糟，全部重装了，所以还是root下比较好。
